Objetivos del Curso

En este curso de introducción a Java aprenderás los conceptos básicos, las mejores técnicas y las herramientas necesarias para programar de forma eficiente con este lenguaje de programación.

¿A quién va dirigido?

Este curso está dirigido a técnicos, tecnólogos y profesionales de la ciencia de la computación interesados en aprender Java para incursionar en el mundo de la programación.
